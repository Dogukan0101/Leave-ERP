# Leave ERP
Leave ERP is a human resources leave tracking web application which ensures to keep a track of employee's leave days.

## Technologies Used:
- Spring Boot
- MySQL
- Redis
- React
- Tailwind CSS
- Docker

## Features Involved:
- Rest API
- Pagination on Backend
- Search on Backend
- Containerization
- Valid Leave Date Check
